# Evidence

## Purpose

This document defines the mechanisms by which assertions within the Universal Information Standard (UIS) are supported or substantiated.

## Scope

The scope addresses:

- Evidence concepts and structure
- Assertion types requiring evidence
- Evidence evaluation criteria
- Evidence chains and aggregation

## Content to be developed

- Evidence definition and purpose
- Assertion and evidence relationships
- Evidence types
- Evidence structure
- Evaluation criteria
- Evidence chains
