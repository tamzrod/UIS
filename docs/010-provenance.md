# Provenance

## Purpose

This document defines how information about the origin, custody, and history of entities is represented within the Universal Information Standard (UIS).

## Scope

The scope addresses:

- Provenance entities (agents, activities)
- Provenance relationships
- Provenance capture and representation
- Provenance quality attributes

## Content to be developed

- Provenance definition and scope
- Agent entities
- Activity entities
- Provenance relationships
- Provenance properties
- Provenance quality
