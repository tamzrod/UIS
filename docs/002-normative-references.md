# Normative References

## Purpose

This document identifies the normative references essential to the Universal Information Standard (UIS).

## Scope

The scope addresses:

- Standards that contain provisions referenced by UIS
- Citation format and applicability
- Status of referenced documents
- Requirement language conventions

## Content to be developed

- List of normative references with citation details
- Applicability statements for each reference
- Interpretation guidance for requirement language
- Reference to RFC 2119 and RFC 8174 for requirement keywords
